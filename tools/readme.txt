Brief instructions for care with Reborn:

- Install Reborn firmware (like any other)
- After installation, the PC will turn on and the installation process will take place
- then turn off the PC and turn it on with 7 pressed, 0xFFing starts
- after the end, turn off, turn on with PTT pressed (firmware mode)
- flash any factory firmware
- using k5prog (version 26), upload the calibration stock to the turned on radio,
- then your saved settings (configuration) - if any.

you can use or install any other non-factory firmware.
